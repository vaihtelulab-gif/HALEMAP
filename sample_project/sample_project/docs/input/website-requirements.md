# 商品紹介サイト（LP）要件定義書

## 1. サイトの目的 (Goals)
1.  **認知・興味喚起:** 「カードを並べてスマホで鳴らす」という新しい体験を、動画やGIFで直感的に伝える。
2.  **ゲムマ集客:** ゲームマーケット2026春（2026/5/23-24）でのブース来場と試遊を促す。
3.  **事前予約:** 当日の取り置き予約フォームへの誘導、およびSNSフォロワーの獲得。

## 2. ターゲット (Target)
- **メイン:** ゲームマーケット参加者（新しいメカニクス好きなゲーマー、トレンドに敏感な層）
- **サブ:** 教育関係者、親御さん（知育玩具として興味を持つ層）

## 3. 必須コンテンツ (Contents)
1.  **ヒーローエリア:** キャッチコピー + プレイ動画（カードをスキャンして音が鳴る瞬間）。
2.  **What is BeatStack?:** 3ステップでの遊び方説明（並べる→撮る→鳴らす）。
3.  **Features:** 「アプリ連携」「動画シェア」「みんなで遊べる」などの特徴紹介。
4.  **Gallery:** プレイ風景の写真、生成された動画のサンプル。
5.  **Game Market Info:** 出展情報（ブース番号、地図）、当日の頒布価格、特典情報。
6.  **Reservation:** 予約フォームへのリンク（Googleフォーム等）/ SNSリンク。

## 4. 技術要件
- **レスポンシブ:** モバイルファースト（スマホ閲覧率90%想定）。
- **パフォーマンス:** 会場回線でも快適に見られる軽量さ（動画は軽量化必須）。
- **OGP:** SNSシェア時に魅力的なカードが表示される設定。

---

## 5. デザイン案

### 案1：Hybrid Pop-Tech（推奨案）
**コンセプト:** 「楽しさ」と「技術」の融合。
中高生やゲーマー層に向けた、ワクワク感と新しさを両立させるデザイン。

- **配色:**
    - ベース: 白 (#FFFFFF) または 薄いグレー (#F5F5F5)
    - メイン: ビビッドなイエロー (#FFD700) または オレンジ (#FF8C00)
    - アクセント: ネオンブルー (#00FFFF) または サイバーグリーン (#00FF00)
    - 文字色: 濃いグレー (#333333)

- **デザイン特徴:**
    - **ヒーローエリア:** 楽しそうに遊んでいる写真の上に、スキャン時の「解析グリッド」や「波形」のアニメーションを重ねる。
    - **タイポグラフィ:** 太めのサンセリフ体（ゴシック体）を使用し、斜体や縁取りでポップさを演出。
    - **あしらい:** 幾何学模様（Tech感）と、手書き風の矢印や吹き出し（Pop感）をミックス。
    - **ボタン:** 立体感のあるボタンや、ホバー時にバウンドするような動きのあるインタラクション。

- **ターゲットへの訴求:**
    - 「難しそう」というハードルを下げつつ、「なんかスゴイことができそう」という期待感を持たせる。

### 案2：Academic Minimal（対抗案）
**コンセプト:** 「知的な遊び」「美しいツール」。
教育関係者やデザイン感度の高い層に向けた、洗練された印象を与えるデザイン。

- **配色:**
    - ベース: 生成り色 (#FAF9F6) または オフホワイト
    - メイン: 濃いチャコールグレー (#2F4F4F)
    - アクセント: 深い赤 (#8B0000) または 落ち着いたゴールド
    - 文字色: 黒に近いグレー (#1A1A1A)

- **デザイン特徴:**
    - **ヒーローエリア:** カードとスマホが美しく配置された静物画のようなメインビジュアル。余白を大きく取る。
    - **タイポグラフィ:** 明朝体やセリフ体（Noto Serif JP等）を使用し、知的な印象に。英語は細身のサンセリフ。
    - **あしらい:** 五線譜をモチーフにした細いラインや、音符記号をさりげなく配置。
    - **ボタン:** フラットでシンプルなデザイン。枠線のみのゴーストボタンなど。

- **ターゲットへの訴求:**
    - 「子供の教育に良さそう」「部屋に置いても邪魔にならない」という安心感と所有欲を刺激する。

## 6. 開発フェーズ
1.  **ティザー公開（2026年3月頃）:** 案1ベースで、メインビジュアルとSNSリンクのみの1枚ペラ。
2.  **本公開（2026年4月下旬）:** 全コンテンツ実装、予約フォーム連携。
