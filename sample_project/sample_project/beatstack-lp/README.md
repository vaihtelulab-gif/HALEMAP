# BeatStack LP

「BeatStack（仮）」の商品紹介ランディングページです。

## セットアップ

このプロジェクトを実行するには、Node.js が必要です。

1. 依存関係のインストール:
```bash
npm install
```

2. 開発サーバーの起動:
```bash
npm run dev
```

3. ブラウザで確認:
`http://localhost:3000` を開いてください。

## 構成
- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Design Concept:** Hybrid Pop-Tech (案1)
